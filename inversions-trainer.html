What is an inversion?

Every chord you've met so far in this app has been played in root position — root on the bottom, then the rest of the chord tones stacked on top in order (3rd, 5th, and for 7th chords, the 7th). An inversion keeps the exact same notes — the same pitch classes, the same chord — but reorders them so a different chord tone sits at the bottom, in the bass. Nothing about which notes belong to the chord changes; only which one is lowest changes.

You build an inversion by taking the note (or notes) currently at the bottom of the chord and moving them up an octave, one at a time, until the tone you want in the bass is the lowest-sounding note. Because the pitch classes are unchanged, an inverted chord is still named after — and still functions as — the very same chord. A C major triad played as E‑G‑C is still a C major chord; it's just wearing a different "shoe" on the bottom.

Inversions matter for two big reasons: smoother voice leading (moving from one chord to the next with the smallest possible hand movement, instead of leaping around the keyboard for every chord change) and bass-line control (choosing exactly which note walks the bass line, rather than always jumping back to the root). Both are essential skills once you start playing real chord progressions instead of isolated chords.

The types of inversion

A chord has as many inversions as it has notes minus one. Every triad (3 notes: major, minor, augmented, diminished) has a root position plus 2 inversions. Every 4-note chord (6th, minor 6th, dominant 7th, diminished 7th, minor 7th, major 7th, half-diminished 7th, augmented 7th, major 7th♭5) has a root position plus 3 inversions:

InversionWhat's in the bassTriad example (C‑E‑G)7th‑chord example (C‑E‑G‑B♭)
Root positionThe rootC – E – GC – E – G – B♭
1st inversionThe 3rdE – G – CE – G – B♭ – C
2nd inversionThe 5thG – C – EG – B♭ – C – E
3rd inversionThe 7th (7th chords only — a triad has no 7th to invert to)—B♭ – C – E – G
Notice the pattern: each inversion just moves the previous bass note up an octave and lets the next chord tone become the new bass note. 1st inversion moves the root up; 2nd inversion moves the root and the 3rd up; 3rd inversion (7th chords only) moves the root, 3rd, and 5th up, leaving the 7th as the lowest note.

In written music, an inverted chord is often shown as a slash chord — the chord name, a slash, then the bass note. C/E means "a C major chord with E in the bass" (1st inversion); C/G means "a C major chord with G in the bass" (2nd inversion); Cmaj7/B means "a C major 7th chord with B — its 7th — in the bass" (3rd inversion).

Building inversions for every chord type explored so far

Because every chord in this app is built from the same idea — a stack of intervals counted in semitones from a root — the exact same 3-step recipe builds an inversion for any of them: (1) start from root position, (2) move the bottom note (or notes) up an octave, (3) the note that's now lowest names the inversion.
